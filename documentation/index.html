1. المقدمة - الأعلى

بسيط هو موقع موقع صفحة واحدة للسيرة الذاتية ومعرض الأعمال تفاعلي، متجاوب بالكامل وقابل للتخصيص، مبني باستخدام Next.js و Tailwind CSS.

الميزات الأساسية:

  • مبني باستخدام React.js و Next.js
  • متجاوب تمامًا وصديق للأجهزة المحمولة
  • Tailwind CSS لتسهيل التخصيص
  • محسن لمحركات البحث (SEO) بفضل تحسينات Next.js
  • أفضل الممارسات مثل:
    • استخدام أداة Cn مع حزم clsx و twMerge
    • استخدام أداة Class Variance Authority (cva) لإدارة أسماء الفئات (CSS) بناءً على شروط مختلفة.
  • جاهز للنشر على منصات مثل Vercel و Netlify وغيرها.

2. التثبيت - الأعلى

المتطلبات الأساسية:

تأكد من تثبيت العناصر التالية على جهازك:

  • Node.js (الإصدار 12.x أو أعلى)
  • npm أو yarn

خطوات التشغيل محليًا:

  1. افتح مجلد المشروع في محرر النصوص الخاص بك.
  2. قم بتثبيت الحزم:
    3. npm install

    أو إذا كنت تستخدم yarn:

    yarn install
  3. تشغيل خادم التطوير:
    4. npm run dev

    أو إذا كنت تستخدم yarn:

    yarn dev

    سيكون الموقع متاحًا على http://localhost:3000.

3. التخصيص - الأعلى

المجلدات الرئيسية:

  • public/: ضع فيه الملفات الثابتة مثل الصور والأيقونات.
  • src/components/: يحتوي على المكونات القابلة لإعادة الاستخدام مثل الهيدر، الفوتر، والأقسام المختلفة.
  • src/app/globals.css: تخصيص إعدادات Tailwind مثل الألوان والتباعد وأحجام الخطوط.
  • src/app/layout: تخصيص البيانات الوصفية (Metadata) لتحسين محركات البحث وتغير الخطوط.
  • src/components/Hero: تخصيص قسم المقدمة (Hero).
  • src/components/About: تخصيص قسم "من نحن".
  • src/constants/*: تخصيص روابط شريط التنقل، أيقونات التواصل الاجتماعي، قسم الخبرات، وقسم الأعمال.

4. النشر - الأعلى

النشر على Vercel:

  1. ادفع (Push) مشروعك إلى مستودع GitHub.
  2. انتقل إلى Vercel، سجل الدخول باستخدام GitHub، وقم باستيراد مستودعك.
  3. سيكتشف Vercel تلقائيًا أنك تستخدم Next.js وسيقوم بإعداد البيئة اللازمة.
  4. بمجرد النشر، سيعطيك Vercel رابطًا حيث سيكون مشروعك متاحًا.

النشر على Netlify:

  1. ادفع (Push) مشروعك إلى مستودع GitHub.
  2. انتقل إلى Netlify، قم بتسجيل الدخول واستيراد مستودعك.
  3. اختر Next.js كإطار العمل.
  4. قم بنشر موقعك.
